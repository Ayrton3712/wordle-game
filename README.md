# Wordle Game Project

This project implements the game Wordle using C++ for the core game logic and Python (Tkinter) for the graphical user interface. Wordle is a trademark of The New York Times Company; this work is for educational use only.

## Structure
- `src/`: Contains all C++ source files
- `include/`: Header files for sharing functions across modules
- `gui/`: Python interface built with Tkinter
